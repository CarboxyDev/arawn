# create-arawn

Create a production-ready full-stack TypeScript application with Next.js, Fastify, and Turborepo.

## Usage

```bash
npm create arawn@latest [project-name]
```

Or with pnpm:

```bash
pnpm create arawn [project-name]
```

## What you get

- ✅ Next.js 16 + React 19 web application
- ✅ Fastify API with Prisma ORM
- ✅ PostgreSQL database with Docker Compose
- ✅ Better Auth for authentication
- ✅ TailwindCSS + shadcn/ui components
- ✅ TypeScript everywhere with Zod validation
- ✅ Testing with Vitest
- ✅ GitHub Actions CI/CD
- ✅ Automated setup script

## Development

Build the CLI:

```bash
pnpm build
```

Watch for changes:

```bash
pnpm dev
```

Type-check:

```bash
pnpm type-check
```

## Publishing

```bash
npm publish
```

The CLI will be available as `create-arawn` on npm.
